# メディア運用方針

## 2つのメディアの位置付け

### デカ太郎ラボ（ブログ）
- **目的**: 収益化
- **URL**: https://dekataro.com
- **集客**: SEO（Google検索）、将来的にPinterest
- **記事の主語**: 読者（「〜の方法」「〜のおすすめ」）
- **テーマ**: 柴犬、DINKs暮らし、お金、グッズ・レビュー、おでかけ、コラム
- **トーン**: ポジティブ、クスッと笑える、誰も傷つけない

### note
- **目的**: フォロワー獲得
- **URL**: https://note.com/dekataro
- **集客**: ハッシュタグ、お題企画、毎日投稿（SNSは使わない）
- **記事の主語**: 自分（「〜した話」「〜を考えた」）
- **テーマ**: キャリア・働き方、趣味（ゴルフ・筋トレ）、内省、柴犬エッセイ
- **トーン**: 等身大、自虐・ユーモア、正直、カジュアル

---

## テーマの棲み分けルール

### 基本原則
**同じテーマでも「主語」で切り分ける。**
- 「自分がどう感じたか」→ note
- 「読者が知りたい情報」→ ブログ

### テーマ別の切り分け

| テーマ | note | ブログ |
|---|---|---|
| 柴犬 | 「序列を突きつけられている話」（体験） | 「豆柴の飼い方まとめ」（情報） |
| 筋トレ | 「挫折した筋トレが今回だけ続いている話」（体験） | 筋トレグッズレビュー（情報） |
| ゴルフ | 「ハマった理由がスコアと関係なかった」（体験） | コース紹介・ギアレビュー（情報） |
| キャンプ | 「キャンプで気づいたこと」（エッセイ） | 「犬連れキャンプ場おすすめ」（ガイド） |
| お金 | **書かない**（職場の人が見る前提） | 家計術・資産形成の情報記事 |
| キャリア | 転職活動のリアル・働き方の内省 | **書かない** |
| 内省 | 言語化・飽き性・挫折の話 | **書かない** |

### 注意事項
- noteには具体的な金額・年収・家計の話を書かない
- ブログには個人的な内省・ポエムを書かない
- noteでは柴犬の名前を出さない（「うちの柴犬」「うちの子」で統一）
- ブログでは「ハチ」の名前を使う

## クロスメディア方針

### 現時点
- 相互リンク・導線はなし
- 完全に独立した2メディアとして運用

### 将来の検討事項
- noteで集めたファンをブログに流す導線（記事末尾にブログリンク）
- noteで書いた体験をブログの冒頭エピソードとして再利用
- ブログ記事のネタをnoteで先行テスト（反応を見てからブログ化）

---

## ネタ管理

### 統一管理の原則
- ネタ出し時に「これはnote向きかブログ向きか」を判断する
- 片方で書いた内容をもう片方で重複して書かない
- ブログのネタ帳: `01_編集部/ネタ帳.md`（ローカル管理）
- noteのネタ帳: `note/NOTE_HANDOVER.md`（コンテンツカレンダー参照）
- 両方のネタ帳を横断して重複チェックする
