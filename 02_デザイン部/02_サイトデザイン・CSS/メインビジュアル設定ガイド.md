# メインビジュアル設定ガイド

## Gemini画像生成プロンプト（チャット画面に貼る）

```
横長（16:9、1600×900px）のイラストを描いてください。

【シーン】
休日の朝、明るいリビングで30代の日本人夫婦がソファに座ってコーヒーを飲んでいる。
夫の膝の上に柴犬が乗っていて、気持ちよさそうにしている。窓から朝日が差し込み、部屋全体が柔らかい光に包まれている。
二人ともリラックスした服装（パジャマやスウェット）で、自然体の笑顔。

【スタイル】
- アニメチック・コミック的なイラスト（線画しっかり）
- ポップで温かみのある色使い
- 日本のおしゃれなライフスタイル雑誌の挿絵のようなテイスト

【色味】
- ベースカラー：クリームホワイト、ベージュ系の暖色
- アクセント：セージグリーン（#5B8C5A）、コーラル（#E8927C）
- 全体的に明るく、柔らかいトーン

【構図の注意点】
- 画像の左側1/3は余白を多めにする（テキストを重ねるため）
- 夫婦は画像の右寄りに配置
- 背景はシンプルに（ごちゃごちゃさせない）
```

## SWELL メインビジュアル設定手順

**WordPress管理画面 → 外観 → カスタマイズ → トップページ → メインビジュアル**

### 1. 基本設定
| 項目 | 設定値 |
|---|---|
| メインビジュアルの表示 | **画像** |
| 表示設定（PC） | **画像サイズのまま** or **数値で指定（50vw）** |
| 表示設定（SP） | **数値で指定（60vh）** |
| Lazy Loadを適用 | OFF |

### 2. スライド画像設定
| 項目 | 設定値 |
|---|---|
| 画像 | Geminiで生成した画像をアップロード |

### 3. テキスト設定
| 項目 | 設定値 |
|---|---|
| メインテキスト | **ふたりだから、ちょうどいい。** |
| サブテキスト | （空欄 or 「子なし共働き夫婦のラクする暮らし」） |
| テキストの位置 | **左** |
| テキストカラー | **#333333** |
| テキストシャドウ | なし（画像の左側が明るい前提） |

### 4. フィルター設定
| 項目 | 設定値 |
|---|---|
| フィルター | **なし** or **ブラー（薄く）** |
| オーバーレイカラー | **#FFFFFF**（白） |
| オーバーレイ透過度 | **0.1〜0.2**（画像が暗い場合のみ） |

### 5. ボタン設定
| 項目 | 設定値 |
|---|---|
| ボタンテキスト | **記事を読む** |
| ボタンリンク先 | `#post_list`（記事一覧へスクロール）or 空欄 |
| ボタンカラー | デフォルト（CSSで制御済み） |

## 画像がまだない場合の暫定対応

Geminiで画像を作るまでの間、以下で対応可能：

1. **単色背景**にする
   - メインビジュアル画像なしで、背景色を `#E8F0E8`（プライマリライト）に設定
   - テキストだけで見せるミニマルなスタイル

2. **フリー素材を仮置き**
   - Unsplashで「couple coffee morning」等で検索
   - あくまで仮なので、早めにオリジナルに差し替え
