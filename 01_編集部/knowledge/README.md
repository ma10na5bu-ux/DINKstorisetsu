# knowledge/ — 文体リファレンス

執筆時にトーン・文体・構成のリズムを参照するための高品質記事コレクション。

## 収録基準
- ふきだし・地の文・構成のバランスが良い
- 2,000〜3,000字の適正範囲
- カテゴリが偏らない（柴犬・DINKs・趣味など）
- 「スクショしてパートナーに送りたくなる」基準を満たす

## 収録記事
1. `2026-04-03_物欲ゼロのDINKs夫婦が唯一お金を使うもの.md` — データ×自虐のバランス
2. `2026-03-30_DINKs夫婦の睡眠事情…md` — ふきだしのテンポが秀逸
3. `2026-04-06_犬が飼いたいで家を買った…md` — フック→展開の構成見本
4. `2026-03-25_DINKs旅行支出は子あり世帯の2倍…md` — データ記事の体験への溶かし方
5. `2026-03-16_奥さんに教わったゴルフ…md` — 趣味系の人物描写

## 使い方
執筆時に「`knowledge/` 内の過去記事を参照し、文体やふきだしのリズムを踏襲して」と指示する。
