# ファビコン生成プロンプト（Gemini Advanced用）

## 生成時の注意
- 生成画像の**右下にGeminiのウォーターマーク**が入るため、アイコンの重要要素は**中央〜左上寄り**に配置する
- ファビコンは16×16〜32×32pxで表示されるため、**細かいディテールは避け、シンプルな形状**を優先する
- 生成後、正方形にトリミングしてからWordPressにアップロードする（推奨: 512×512px）

---

## プロンプト

```
正方形のファビコン用アイコンを生成してください。

【デザイン】
- メインモチーフ：柴犬の顔を正面からシンプルに描いたアイコン
- 丸みのあるデフォルメされた柴犬の顔（リアルではなくアイコン的な簡略化）
- 柴犬の表情：にっこり笑っている、親しみやすい顔
- モチーフは画像の中央〜やや左上に配置すること（右下にウォーターマークが入るため）

【色】
- 柴犬の毛色：暖かいベージュ〜薄茶（赤柴カラー）
- 背景：セージグリーン（#5B8C5A）の円形背景、またはセージグリーンのベタ背景
- 柴犬の頬・耳の内側にコーラル（#E8927C）をさりげなく使用

【スタイル】
- フラットデザイン（グラデーション・影・3D効果なし）
- 線は太め（小さく表示しても視認できるように）
- 細かい装飾・テキスト・文字は一切入れない
- 32×32pxに縮小しても「柴犬の顔」と判別できるシンプルさを最優先

【構図】
- 正方形キャンバス（1:1比率）
- アイコンはキャンバスの中央に大きく配置
- 余白は上下左右に均等に10〜15%程度
- 右下20%のエリアには重要な要素を置かない（ウォーターマーク対策）

【NG】
- テキスト・文字・ロゴタイプは入れない
- 人物・キャラクターは入れない（柴犬のみ）
- 複雑な図形・細い線・グラデーションは使わない
- リアルな犬の描写にしない（アイコン的なデフォルメで）
```

## バリエーション案

必要に応じてモチーフを差し替えて再生成：

| モチーフ案 | イメージ |
|---|---|
| 柴犬の顔（正面） | DINKs＋犬の暮らし（メイン案） |
| 家の屋根＋柴犬のシルエット | おうち×犬のシンボル |
| 柴犬の肉球 | シンプル・可愛い・犬モチーフ |

---

## トリミング手順

1. 生成された画像を開く
2. 右下のGeminiウォーターマーク部分を避けて正方形にトリミング
3. 512×512pxにリサイズ
4. WordPress管理画面 → 外観 → カスタマイズ → サイト基本情報 → サイトアイコンに設定
