# DINKsのトリセツ デザインガイドライン

## コンセプト

「ポジティブなラクする暮らし」を視覚的に体現するデザイン。
温かみがありつつも洗練された、大人カップルのライフスタイル感。

---

## カラーパレット

### メインカラー
| 名前 | HEX | 用途 |
|---|---|---|
| Primary（セージグリーン） | `#5B8C5A` | ヘッダー・ボタン・見出し装飾 |
| Primary Dark | `#3D6B3C` | ホバー・強調 |
| Primary Light | `#E8F0E8` | 背景アクセント・カード背景 |

### アクセントカラー
| 名前 | HEX | 用途 |
|---|---|---|
| Accent（コーラル） | `#E8927C` | CTA・リンクホバー・アイコン |
| Accent Light | `#FFF0EC` | 注目エリアの背景 |
| Sub Accent（マスタード） | `#D4A843` | タグ・バッジ・ワンポイント |

### ベースカラー
| 名前 | HEX | 用途 |
|---|---|---|
| Background | `#FDFBF7` | サイト全体の背景 |
| Surface（カード背景） | `#FFFFFF` | 記事カード・コンテンツエリア |
| Text（本文） | `#333333` | 本文テキスト |
| Text Light | `#666666` | 補足テキスト・日付 |
| Border | `#E5E0DA` | 区切り線・カード枠 |

### カラー選定理由
- **セージグリーン**：自然体・リラックス・ポジティブ。DINKsの「自由な暮らし」を象徴
- **コーラル**：温かみ・親しみ・夫婦の柔らかな関係性
- **マスタード**：遊び心・アクセント。「クスッと笑える」要素の視覚表現
- **クリームホワイト背景**：柔らかく目に優しい。長文記事でも読み疲れない

## アイキャッチ画像

### 基本方針
- **イラストのみ（文字なし）を基本**とする
- 文字入れはSNS用OGPが必要な場合のみ、別途Canva等で作成
- Geminiの画像生成で日本語テキストを入れると品質が不安定なため

### 仕様
| 項目 | 値 |
|---|---|
| サイズ | 1200×630px（OGP兼用） |
| スタイル | アニメチック・コミック的なイラスト |
| テイスト | ポップで温かみのある色使い |
| 生成ツール | Gemini Advanced（チャット画面） |

### 3パターンの使い分け

| パターン | 使うタイミング | 構図のポイント |
|---|---|---|
| **シーン型** | 体験記事・休日記事 | 夫婦の日常シーンを描く。場所や状況が伝わる構図 |
| **あるある型** | DINKsあるある記事 | 1コマ漫画風。「ツッコミどころ」がある構図。ヨメ氏の反応を入れる |
| **アイコン型** | 知識・まとめ記事 | シンプル背景＋テーマを象徴するアイテムを中央に配置 |

### 差別化（vs 柴犬といっしょ）
| | 柴犬といっしょ | DINKsのトリセツ |
|---|---|---|
| スタイル | リアルな柴犬写真 | アニメ・コミック調イラスト |
| 主役 | 柴犬 | 夫婦の日常シーン |
| テイスト | 写実的・可愛い | ポップ・おしゃれ・クスッと笑える |

### キャラ描写の統一ルール
| キャラ | 外見の特徴 | 表情・ポーズ |
|---|---|---|
| デカ太郎 | 黒髪短髪、30代、普通体型 | 楽しそう・ドヤ顔・やらかしてる |
| ヨメ氏 | ボブカット茶髪、30代 | 呆れ顔・冷静・優しい笑顔 |
| 柴犬 | 赤柴、中型 | 自由気まま・マイペース |
