# アイキャッチ画像 Geminiプロンプト集

> Gemini Advancedのチャット画面に貼って画像生成する。
> `{{変数}}` の部分を記事に合わせて書き換えて使う。

---

## 共通の基本設定（全パターン共通の末尾ブロック）

以下はすべてのプロンプトの末尾に付ける。コピペ用。

```
【スタイル】
- アニメチック・コミック的なイラスト（線画しっかり）
- ポップで温かみのある色使い
- 日本のおしゃれなライフスタイル雑誌の挿絵のようなテイスト

【色味】
- ベースカラー：クリームホワイト、ベージュ系の暖色
- アクセント：セージグリーン（#5B8C5A）、コーラル（#E8927C）
- 全体的に明るく、柔らかいトーン

【共通ルール】
- 文字やテキストは一切入れないでください
- 画像内にウォーターマークやロゴを入れないでください
- 重要な要素（人物・ペット・キーアイテム）は画像の中央寄り（左右の端から20%以内に配置しない）に収めてください。上下左右をトリミングする可能性があるため、端ギリギリに重要な要素を置かないでください
```

## パターン1：シーン型（体験記事・休日記事）

```
横長（16:9、1200×630px）のイラストを描いてください。

【シーン】
{{場所・状況の説明}}
30代の日本人夫婦が{{何をしているか}}。
{{柴犬の様子（任意）}}

【構図の注意点】
- 場所の雰囲気が伝わるように背景もしっかり描く
- 夫婦の自然な表情やしぐさを大切に
- ごちゃごちゃさせず、何の場面かひと目でわかる構図
```

#### 使用例：「日曜の朝、夫婦で『今日どうする？』から始まる最高の一日」

```
横長（16:9、1200×630px）のイラストを描いてください。

【シーン】
日曜の朝、明るいリビングで30代の日本人夫婦がソファに座っている。
二人ともパジャマ姿でコーヒーを持ち、「今日どうする？」と相談している雰囲気。
テーブルの上にはスマホとガイドブックが散らばっている。
柴犬がソファの横で丸くなって寝ている。

【構図の注意点】
- 場所の雰囲気が伝わるように背景もしっかり描く
- 夫婦の自然な表情やしぐさを大切に
- ごちゃごちゃさせず、何の場面かひと目でわかる構図
```

## パターン2：あるある型（DINKsあるある記事）

```
横長（16:9、1200×630px）のイラストを描いてください。

【シーン】
{{あるあるの場面を具体的に描写}}
30代の日本人男性（夫）が{{やらかしている行動}}。
妻が{{リアクション（呆れ顔・ツッコミ顔・冷静に見てる等）}}。

【構図の注意点】
- 1コマ漫画のような「ツッコミどころがある」構図にする
- 夫の行動と妻のリアクションの対比が面白さのポイント
- 「これ、うちもだ！」と思わせる生活感のあるディテール
```

#### 使用例：「うちの2LDK、寝室以外ぜんぶ趣味部屋です」

```
横長（16:9、1200×630px）のイラストを描いてください。

【シーン】
2LDKマンションの一室が完全にキャンプ用品の趣味部屋になっている。
テントが部屋の隅に立てられ、棚にはランタン・バーナー・シェラカップが並んでいる。
壁にはアウトドアブランドのステッカーが貼ってある。
30代の日本人男性（夫）が嬉しそうにキャンプギアを眺めている。
ドアの外から妻が呆れた顔で覗いている。

【構図の注意点】
- 1コマ漫画のような「ツッコミどころがある」構図にする
- 夫の行動と妻のリアクションの対比が面白さのポイント
- 「これ、うちもだ！」と思わせる生活感のあるディテール
```

## パターン3：アイコン型（知識・まとめ記事）

```
横長（16:9、1200×630px）のイラストを描いてください。

【シーン】
シンプルな背景（薄いベージュ or クリームホワイトの無地）の中央に、
{{テーマを象徴するアイテム・モチーフ}}がおしゃれに配置されている。
周囲に{{関連する小物を3〜4個}}が散りばめられている。

【構図の注意点】
- 雑誌の特集ページの扉絵のようなすっきりした構図
- アイテムが主役。人物は入れない
- 余白を活かしたミニマルなレイアウト
```

#### 使用例：「うちの家計簿、『趣味費』の欄だけ異常に太い」

```
横長（16:9、1200×630px）のイラストを描いてください。

【シーン】
シンプルな背景（薄いベージュの無地）の中央に、
開いた家計簿が置かれている。「趣味費」の欄だけ数字がびっしり書き込まれていて太い。
周囲にクレジットカード、レシートの山、コーヒーカップが散りばめられている。

【構図の注意点】
- 雑誌の特集ページの扉絵のようなすっきりした構図
- アイテムが主役。人物は入れない
- 余白を活かしたミニマルなレイアウト
```

## Geminiマークの除去

生成後、右下にGeminiのウォーターマークが入る場合がある。
Claude Codeに画像パスを伝えれば自動でトリミング処理する。
